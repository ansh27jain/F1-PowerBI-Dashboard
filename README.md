#  Formula 1 Performance Dashboard (Power BI)

##  Overview
This project is a 4-page interactive Power BI dashboard analyzing Formula 1 data from 1950 to 2024. It provides insights into driver performance, constructor dominance, and circuit trends.

---

##  Key Insights
- Ferrari leads historically in total points  
- Lewis Hamilton is the highest point scorer  
- Red Bull shows strong dominance in recent years  

---

##  Dashboard Sections

###  Overview
- Total races, drivers, constructors, and circuits  
- High-level performance summary  

###  Driver Performance
- Total points, wins, and podium finishes  
- Comparison of top drivers across seasons  

###  Constructor Performance
- Team dominance (Ferrari, Mercedes, Red Bull)  
- Wins, podiums, and total points analysis  

###  Circuit Analysis
- Race distribution across circuits  
- Circuit-wise trends and insights
